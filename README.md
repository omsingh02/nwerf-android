# Nwerf (Main Branch)

Welcome to **Nwerf**, a private, serverless music engine built with Kotlin and Jetpack Compose.

This branch (`main`) contains the core stable features of the Nwerf application.

## Core Features

- **Audio Identification (Aura)**: Powered by ACRCloud, instantly identify songs playing around you.
- **Continuous Mode**: Continuously listen and identify tracks without pressing a button.
- **Auto-Download**: Automatically fetch identified tracks and upload them to your private Telegram storage.
- **Telegram Vault**: All your music is stored and streamed directly from your private Telegram channel, meaning unlimited, free, and secure cloud storage.
- **Persistent History**: Keeps a local record of your previously identified tracks so you never lose a song.
- **Modern UI**: Built entirely with Jetpack Compose featuring a sleek, responsive, and dark-mode optimized interface.

## Setup

To run this version, you will need:
1. A Telegram Bot Token
2. A Telegram Chat ID (Your private channel)
3. ACRCloud Credentials (configured in `local.properties`)

## Development

This is the `main` stable branch. For the latest experimental features including Firebase Cloud Sync and advanced CI/CD pipelines, please switch to the `dev` branch!
